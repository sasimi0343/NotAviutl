# 各スクリプト
## ExtremeTransition
個別オブジェクトの登場用に作りました。
アニメーション効果の末端に"@ET"がついているものがこのスクリプトです。

### 共通パラメータ
[Global] が先頭についているものはExtremeTransition内共通のパラメータです
* [Global]Delay
  * 個別オブジェクトの番号順にずらして登場させる時の遅延(フレーム数)です。
  * 例えばDelayが2の場合、番号1のオブジェクトの2フレーム後に番号2のオブジェクトが登場します。
* [Global]Time
  * 登場にかかる時間(フレーム数)です。
  * イージングにかかる時間に影響します。
* [Global]GlobalDelay
  * 全体にかける遅延(フレーム数)です。
* Easing
  * イージングの番号です。 **正の数であればCurve Editorの番号** 、 **負の数であればイージングの番号** になります。

(注意: 「Move@ET」のStart座標が反転しています。)
(注意: 「FanClipping@ET」の使用にはrikkyさんの「扇クリッピング」のスクリプトから「"fan_clipping.lua"」をExtremeTransitionのフォルダに入れる必要があります。)

### 追記 (2024/03/14)

## MoreShapes
さまざまな図形やオブジェクトを追加するために作りました。
また、MoreShapes3Dでは、3Dオブジェクトの描画ができるようにしています。
図形は「カスタムオブジェクト」に入っています。

(比較的動作は不安定な方です)

### 追記 (2024/06/29)
[RandomFigures]を追加しました。
設定ダイアログの「Figures」項目に入力された特定の数値に対応した図形が
ランダムに選出され、表示されます。
例:{1,5,7}

### MoreShapes3Dの使い方 (基本)
1. テクスチャとなるメディアオブジェクトを追加します。(四角形や画像など)
2. オブジェクトを拡張描画にし、カメラ制御内に入れます。
3. アニメーション効果「[3DShape]Box」を追加します。(立方体になります。)
   * 球なら「[3DShape]Ball」
   * n角錐なら「[3DShape]Corn」
   * 直方体は「[3DShape]BoxEx」
4. 必要に応じて追加で効果を加えます。
5. 最後に「[3D]Render」を加えます。

(一部機能の使用には「GetColor」の「getcolortools.lua」をMoreShapesフォルダ内に入れる必要があります。)

## NotAviutl
様々なエフェクト、複雑なカスタムオブジェクトを追加します。
アニメーション効果、カスタムオブジェクトの名前の接頭辞によって種類があります。
(アニメーション効果を"anm"、カスタムオブジェクトを"obj"とします。)

| 接頭辞 | 種類 | 説明 |
| ------ | ---- | ---- |
| [Audio] | anm | (2024/08/15追加) |
| [Camera] | obj/cam | ※使用非推奨、カメラの位置を部分的に変えようとしました。 |
| [Effect] | anm | エフェクト全般、一部動作不安定なものがあります。 |
| [Glitch] | obj | グリッチ系を扱います。一種類のみ。 |
| [Graph] | anm | (2025/04/14追加)様々なグラフを作成します。 |
| [Group] | anm/obj | グループを扱います。 **~~使用方法は後述します。~~** |
| [Liquid] | anm | 液体風のエフェクトをかけます。一種類のみ。 |
| [MathShape] | obj | (2025/04/14追加)数式を描画します。まだ動作が不安定です。 |
| [Motion] | anm | 動きに合わせたエフェクトをかけます。一種類のみ。 |
| [Objects] | anm | (2025/04/14追加)個別オブジェクトのうち特定のもののみを操作するスクリプトです。 |
| [Path] | anm | 他のソフトにありがちな"パス"という概念を追加します。 |
| Pen/NewPen | anm | ペンです。 |
| [Prefab] | obj/anm | オブジェクトを後のフレームから呼び出せます。 |
| [Screen] | obj | 画面全体にかけるエフェクトを追加します。 |
| [Sheet] | anm | (2025/04/14追加)表を描画します。 |
| [Text] | anm | テキスト用に作りました。(ControlPadの使用は非推奨) |
| [Utility] | anm | あったら便利だなというノリで作りました。 |
| [Value] | obj | トラックバーで変数を操作します。スクリプト制御や、アニメーション効果のパラメータ設定などで使用する用です。 |
